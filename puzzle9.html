Calculus problem — find the integer \(D\)

Variables: A = 1225, B = 23, C = 75

Problem

Let 

f(x) = A \ln\big(Bx + C\big)

Find the integer D such that the slope of the tangent to \(f\) at \(x=D\) equals \(B\); that is, 

f'(D) = B.
Hint (first step only)
Compute the derivative \(f'(x)\) using the chain rule and simplify your expression in terms of \(A,B,C,x\).

Solution (click only if you want to reveal it):
  1. Differentiate: \(f'(x)=\dfrac{AB}{Bx+C}.\)
  2. Set \(f'(D)=B\): \(\dfrac{AB}{BD+C}=B.\)
  3. Solve: \(\dfrac{A}{BD+C}=1\Rightarrow BD+C=A.\)
  4. Therefore
